Hello everyone! I’m an undergraduate student of the software engineering course. Here is my list of lab and project journey to build up the “AduKafe” app that functions as a real-time price-ceiling checker and an interactive reporting system.

SDG 12: Responsible Consumption and Production

Core Problem: We are tackling food price manipulation and the lack of price ceiling transparency across campus cafeterias, which unfairly burdens university students. AduKafe empowers students to report and monitor these issues transparently.

Lab Progress

Lab 1: UI Fundamentals

This lab focused on the fundamentals of building a user interface using Jetpack Compose. I learned how to structure layouts with Columns and Rows, and apply basic modifiers to create a clean, responsive screen.

Lab 2: Interaction

In this lab, I explored user interaction by implementing clickable elements and state management mechanisms. Managing state effectively allowed the app UI to dynamically update in response to user inputs.

Lab 3: Material Design

I applied Material Design 3 principles to enhance the visual appeal of the app. Utilizing Material components like Cards, Scaffolds, and consistent typography significantly improved the overall user experience.

Lab 4: Navigation

This session covered navigation within a Jetpack Compose application using NavHost. I successfully implemented multi-screen flows, allowing seamless transitions between the dashboard, forms, and detail screens.

Lab 5: Room Database

I integrated the Room database to handle local data persistence efficiently. This ensures that user profiles and complaint histories remain securely saved on the device even after the app is completely closed.
